# 部门管理

## 查询部门列表

### 基本信息

> 请求路径：/api/departments
>
> 请求方式：GET
>
> 接口描述：该接口用于查询部门列表数据（不分页），根据最后操作时间倒序。

#### 请求参数

参数格式：queryString

参数说明：

| 参数名称 | 是否必须 | 示例 | 备注 |
| -------- | -------- | ---- | ---- |
| deptName | 否       | 学工 | 部门名称，模糊查询 |

请求参数样例：

```shell
/api/departments
/api/departments?deptName=学工
```

#### 响应数据

参数格式：application/json

参数说明：

| 参数名              | 类型      | 是否必须 | 备注                           |
| ------------------- | --------- | -------- | ------------------------------ |
| code                | number    | 必须     | 响应码，1 代表成功，0 代表失败 |
| msg                 | string    | 非必须   | 提示信息                       |
| data                | object[ ] | 必须     | 返回的数据                     |
| \|- id              | number    | 必须     | 部门ID                         |
| \|- deptName        | string    | 必须     | 部门名称                       |
| \|- lastOperateTime | string    | 必须     | 最后操作时间                   |

响应数据样例：

```json
{
  "code": 1,
  "msg": "success",
  "data": [
    {
      "id": 1,
      "deptName": "学工部",
      "lastOperateTime": "2026-03-20 14:57:04"
    },
    {
      "id": 2,
      "deptName": "教研部",
      "lastOperateTime": "2026-03-20 14:57:04"
    }
  ]
}
```

## 根据ID查询部门

> 请求路径：/api/departments/{id}
>
> 请求方式：GET
>
> 接口描述：该接口用于根据ID查询部门数据（编辑回显）。

#### 请求参数

参数格式：路径参数

参数说明：

| 参数名 | 类型   | 是否必须 | 备注   |
| ------ | ------ | -------- | ------ |
| id     | number | 必须     | 部门ID |

请求参数样例：

```shell
/api/departments/1
```

#### 响应数据

参数格式：application/json

参数说明：

| 参数名              | 类型   | 是否必须 | 备注                           |
| ------------------- | ------ | -------- | ------------------------------ |
| code                | number | 必须     | 响应码，1 代表成功，0 代表失败 |
| msg                 | string | 非必须   | 提示信息                       |
| data                | object | 非必须   | 返回的数据                     |
| \|- id              | number | 非必须   | 部门ID                         |
| \|- deptName        | string | 非必须   | 部门名称                       |
| \|- lastOperateTime | string | 非必须   | 最后操作时间                   |
